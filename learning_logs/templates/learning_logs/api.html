API Documentation

Access to All Topics

To access all topics, make a GET request to the following endpoint: /api/

Access to Most Liked Topics

To access the most liked topics, make a GET request to the following endpoint: /api/most_liked_topics/

Access to Each Topic Individually

To access a specific topic, replace {topic_id} with the topic's ID in the following endpoint: /api/{topic_id}/

Follow User (For Authenticated Users)

Note: Authentication is required for this endpoint.

To follow a user, make a POST request to the following endpoint: /api/follow/{user_id}/

My profile(For Authenticated Users)

Note: Authentication is required for this endpoint.

To access personal profile, make a POST request to the following endpoint: /api/my_profile/

Followed profiles(For Authenticated Users)

To acces user profile, make a POST request to the following endpoint: /api/followed_profiles/

User profile

To acces user profile, make a POST request to the following endpoint: /api/user_profile/{user_id}/
